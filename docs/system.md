# Sistema – App de Buceo

## Objetivo

Aplicación web para registrar y consultar inmersiones.

## Usuarios

- Buceadores recreativos

## Funciones críticas

- Ver inmersiones
- Crear inmersión

## No objetivos (por ahora)

- Carga desde ordenador de buceo
- Generación de contenido por IA

## Riesgos conocidos

- Caída del backend
- Latencia alta
- Pérdida de datos
